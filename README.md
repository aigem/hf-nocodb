# hf-nocodb
### 在HuggingFace上部署NocoDB，开源版本的AirTable 。支持S3数据持久化，部署简单，部署后数据还在。我们使用Cloudflare R2作为S3对象存储。

## 一键部署【推荐】
## 在本项目的Spaces中，点击右上角[复制Spaces]按钮. Done
